Data privacy statement

This data privacy statement explains how and what personal data we collect from you through our website.

Personal data means all data that can relate to you personally, such as your name, address, e-mail account(s), user behaviour.

We do not track by default

By default, this website do not track your data. It uses the opt-in principle.

If you choose "No, let me visit this site without cookies / tracking" or you don't make a choice (meaning that you still see the cookie consent banner), the website won't track anything.

But you won't have any restrictions when viewing the content.

gtag('consent', 'default', { ad_storage: 'denied', analytics_storage: 'denied', });

If you choose "Yes, I agree" on the cookie consent notice, tracking using Google Analytics will be activated.

gtag('consent', 'update', { ad_storage: 'granted', analytics_storage: 'granted', });

Which data are tracked by Google Analytics?

This website use Google Analytics to collect some data including: pages viewed, rebound rate, device used.

This site doesn't send your IP adress to Google Analytics.

Because it doesn't comply with the General Data Protection Regulation (GDPR) (EU) 2016/679 , the collect of data is anonymized.

So we use the following tag on each page:

gtag('config', '<GA_MEASUREMENT_ID>', { 'anonymize_ip': true });

Cookies policy / Cookie consent

Cookies are usually small text files, given ID tags that are stored on your computer's browser directory or program data subfolders.

Cookies are created when you use your browser to visit a website that uses cookies to keep track of your movements within the site, help you resume where you left off, remember your registered login, theme selection, preferences, and other customization functions.

Use of cookies is regulated by the E-privacy Directive 2009/136/EC . That's the reason of cookie consent notice.

The cookie consent is displayed on each static page of this website including homepage and landing pages, even if you don't reach this website with the homepage. It will be removed only after approval or rejection.

This website doesn't use cookies. But Google Analytics could use some - if cookie consent has been accepted.

Cookie Name Expiration Time Description
_ga 2 years Used to distinguish users.
_gid 24 hours Used to distinguish users.
_gat 1 minute Used to throttle request rate. If Google Analytics is deployed via Google Tag Manager, this cookie will be named _dc_gtm_<property-id>.
Source: Google Analytics Cookie Usage on Websites

We do not have any access to the cookies delivered by Google Analytics on your computer.
