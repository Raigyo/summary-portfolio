Summary

Group work to learn how to use node.js (server-side) and websockets (Socket IO) with an online game.

  • Node.JS (server-side)
  • Websockets/Socket IO (server-side)
  • React JS (client-side)

The design is inspired by old arcade games.

Many css features are used like css animations with sprites sheets, crt blinking effect and scanlines...

Rules are simple, it's a Rock, Paper, Scissors, Lizard, Spock game. 

You can create a room and another player can join it and... fight!
