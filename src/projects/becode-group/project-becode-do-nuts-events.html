Summary

CRUD Application to manage events online like Eventbrite.

It's a fork of the work of another group.

We had to respect the original layout, correct bugs and non functionnal features then add new features.

  • Laravel - PHP Framework
  • React - JS Framework
  • Postgresql - Database

You need to register / login, then you could add cultural events and suscribe to existing ones. Features that were added are the possibility to add videos and displaying a map of the place of the event.
