Summary

This is the final lab carried out at Technifutur. The aim was to create an application concept and carry out a functional analysis.

It includes :

      - Charter Project
      - Specifications (fr: Cahier des charges) - The main document used during all the project
      - BPMN AS-IS
      - PMN TO-BE
      - Entity Relationship Diagram
      - Database relational Schema
      - UML Activity Diagram
      - UML Class Diagram
      - UML Use Cases Diagrams
      - UML Use Cases Scenarii
      - Wireframes (Balsamiq)
      - Mockup for the homepage (Figma)
